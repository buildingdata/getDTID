### 简介

本软件以考虑太阳辐射热作用的动态室外计算参数作为室外边界条件，针对不同地区不同房间采暖运行方式，基于有限差分法建立了围护结构动态传热模型并提出动态保温设计计算方法，从而获得室内空气温度和各朝向围护结构内表面温度的逐时值，进而依据不同设计指标判定围护结构保温性能是否在各时刻满足防结露与基本热舒适要求，为不同地区围护结构实现精细化动态保温设计提供分析工具，提高建筑的气候适应性。
